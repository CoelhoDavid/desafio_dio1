História da computação

A história da computação começa muito antes da criação dos primeiros computadores eletrônicos. Na antiguidade, os humanos desenvolveram métodos para contar e realizar cálculos simples. Os ábacos, dispositivos mecânicos de cálculo, foram usados na China e na Mesopotâmia há milhares de anos. No entanto, um dos primeiros marcos notáveis foi o uso do ábaco romano, que permitia realizar operações aritméticas básicas.

Charles Babbage inventou o primeiro computador programável do mundo

Charles Babbage (1791-1871) nasceu numa família abastada - era filho de um banqueiro - e se tornou engenheiro mecânico e matemático. Formado pelo Trinity College (em Cambridge), em 1822 desenvolveu o projeto de um dispositivo eletrônico (uma máquina analítica que serviria de base para os computadores), mas que não chegou propriamente a ser concluído. Apesar de ser pouco conhecido pelo grande público, Babbage foi o criador do primeiro c omputador programável do mundo. Em 1823 chegou mesmo a receber verba do estado britânico para criar uma máquina capaz de resolver todos os tipos de cálculos. Babbage chamou o seu invento de Difference Engine, os seus esboços estão disponíveis no Museu de Ciência de Londres. Ada Augusta King, a condessa de Lovelace (filha do poeta inglês Lord Byron) foi uma pioneira, matemática e escritora que, encantada pelo invento de Charles, fez os primeiros programas para a invenção de Babbage. Ada é considerada a primeira programadora da história por ter escrito um algoritmo compatível.

Client e Server

Client (Cliente)

O cliente é quem faz o pedido. Normalmente é o navegador (Chrome, Firefox, etc.) que você está usando.

Examplos:

Quando você digita www.google.com, o navegador:

  1. Faz uma requisição para o servidor do Google
  2. Espera a resposta
  3. E exibe a página pra você

Server (Servidor)

O servidor é quem responde ao pedido. Ele fica "ouvindo" as requisições e devolve o que foi pedido — normalmente arquivos HTML, CSS, JS, ou dados de um banco de dados.

Exemplo:

O servidor do Google recebe o pedido que seu navegador fez e responde com:

  1. O HTML da página de busca
  2. Arquivos CSS pra estilizar
  3. Scripts JS pra funcionar

Basicamente cliente somos nós em nossos dispositivos acessando um site que está hospedado em um servidor(Um computador). E nós como cliente fazemos requisições para os servers.

Tags

Tags são os blocos de construção do HTML. Elas dizem ao navegador como exibir o conteúdo na página.

São exemplos de tags:

p - Paragrafo

h1,h2,h3... - São Títulos

Prodemos usar o href para fazer navegação entre páginas

Também temos as tags de lista ordenada (ol) e não ordenada (ul).
